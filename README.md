This project presents an interactive Insurance Analytics Dashboard designed to help insurance companies track key business metrics such as premium amounts, claims, customer demographics, policy activity, and age-wise claim distribution.

The dashboard provides a consolidated view of business performance and supports data-driven decision-making for stakeholders.
